Programming

I chose to expand on the programming knowledge I obtained from engineering and attend App Academy, a 16 week immersive coding bootcamp. This gave me the opportunity to complete over 1000 hours of hands-on software development and many projects.

Projects

interView

interView is a video platform application where you can practice your interviewing skills and elicit feedback. interView seeks to make the interviewing process easier for job seekers. I was the backend lead utilizing the MERN stack and AWS S3 cloud services.

OurTube

OurTube is a youtube clone that allows users to upload and view videos uploaded by other users. Users can create thier own profiles and view the profiles of other users. While viewing a video users have the ability to comment on, like, or dislike the video. Additionally users can see if other users have liked, disliked, or commented on a video. I built this project from the backend to the frontend and everything in-between.

Another Survival

An interactive survival game programmed using javascript and rendered using html canvas. Another Survival combines third-person overhead perspective, pixel art, and survival gameplay. I built this project from the backend to the frontend and everything in-between.
